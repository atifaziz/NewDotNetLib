# NewDotNetLib

This is [my][me] template repository for setting up new .NET Core class
library projects quickly.

1. To set up the new project, run:
   - `clone.cmd <DIR>` on Windows, or
   - `./clone.sh <DIR>` on Linux/macOS.
2. Review CI configuration in `appveyor.yml`.
